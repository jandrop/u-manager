# U-Manager

Mobile app for managing and monitoring your Unraid server.

## Download
Download from [Google Play](https://play.google.com/store/apps/details?id=es.atrapandocucarachas.umanager)

## Support

Found a bug or have a feature request? [Open an issue](https://github.com/jandrop/u-manager/issues/new).

Before reporting, please include:
- App version (Settings > About)
- Unraid version
- Steps to reproduce the issue

## Requirements

- Unraid 6.12+ with [API enabled](https://docs.unraid.net/API/)
